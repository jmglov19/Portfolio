PokerBot



The Purpose

The PokerBot project is something I started back in 2023 for my senior seminar project. Its purpose was to be good enough to beat a bot playing randomly and a bot using a simple choice system by using the Effective Hand Strength Algorithm (EHS) to look at all possible chances of what its opponent could have. The type of poker used was 1v1 Texas holdâ€™em. This was a resounding success but recently my next step is to bring this bot into more o f a world scenario.

The Execution

Instead of the bot just playing against other low level and low thinking bots I wanted it to be able to beat humans and other challenging bots. I added a GUI through Tkinter that allows the user to play against the bot with also a DEV mode that allows the user to see both hands. This program uses multithreading in python to allow the player to interact with the GUI while the bot makes choices and the game runs. This project is in a stable phase to play against the bot but the bot is still being improved to better react to bluffs and to play more like a human.
