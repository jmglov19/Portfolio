Dijikstra Algorithm Viewer



The Purpose

The purpose of this web app is to be a visual representation of Dijkstraâ€™s shortest path algorithm. This algorithm finds the shortest distance between a start and end node trying all possible paths showing that sometimes the most straightforward path is not always the fastest. The user inputs a start node and an end node then the program finds the shortest algorithm and then returns the shortest path and visually shows the user on the page.

The Execution

The Dijkstra Algorithm Viewer is a web app that allows the user to place nodes on the page and then connect them, assigning a distance of how far each point is. This is not a visual representation of distance but a distance that is set to each connection between each node. This allows the user to set any distance.


This web app was co-developed for the final project in Graph Theory at St. Lawrence University with Phillip Maphey. Try it out below!
