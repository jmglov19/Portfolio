Road Of Champions



The Purpose

This is a game I’ve been developing in the Godot engine as a 2d Platformer. The idea started as a choose your own adventure based in java where you just clicked buttons on what you wanted to do next but this wasn’t as fun as I wanted. So I used the same concepts and some of the same code ideas and made a 2d Platformer. Godot is a great intro game development tool and was great for what I wanted to build.

The Execution

I have been able to get many of the assets I need through free downloads but for some animations like the bow animation I have had to draw it myself with pixels in photoshop. This part has been some of the most difficult but also the most fun to see it come to life in the game.
